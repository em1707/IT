Chapter: 1

Chapter Summary

A computer is an electronic device that processes data (unprocessed items like text and numbers) into information (processed, useful data). This process follows the information processing cycle: input, processing, output, and storage. While computers offer significant advantages like speed, reliability, and vast storage, they also present disadvantages such as health risks, privacy concerns, and environmental impact. A complete information system involves five elements: hardware, software, data, people, and procedures. Computers are now integral to almost every part of society, including education, finance, government, health care, and manufacturing.

Chapter: 2

Chapter Summary

A computer's physical parts, which you can see and touch, are called hardware. The instructions that tell the hardware what to do are called software. There are five main categories of hardware components: input devices (e.g., keyboard, mouse), output devices (e.g., monitor, printer), the system unit (which houses the processor and memory), storage devices (e.g., hard disk, USB drive), and communications devices (e.g., modem). Software is broadly divided into two types: system software (like an Operating System - Windows, macOS) which controls the computer, and application software (like a web browser or word processor) which allows users to perform specific tasks.
